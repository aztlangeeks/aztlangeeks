1.- Ejercicio fdp01_t02_01: ParImpar.java

Escriba una aplicación que lea un entero y que determine e imprima si es impar o par [sugerencia: use el operador residuo. Un número par es un múltiplo de 2. Cualquier múltiplo de 2 deja un residuo de 0 cuando se divide entre 2].

// Ejercicio fdp01_t02_01: ParImpar.java
// Aplicación que lee un entero y que determina e imprime si es impar o par.
package t02_01;

import java.util.Scanner;

public class ParImpar {

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        
        int numero;
        
        System.out.print("Escriba un número entero: ");
        numero = sc.nextInt();
        
        if (numero % 2 == 0)
            System.out.println(numero + " es un número par.");
        
        if (numero % 2 == 1)
            System.out.println(numero + " es un número impar.");
    }
    
}

2.- Ejercicio fdp01_t02_02: Multiplos.java

Escriba una aplicación que lea dos enteros, determine si el primero es un múltiplo del segundo e imprima el resultado.

// Ejercicio fdp01_t02_02: Multiplos.java
// Aplicación que lea dos enteros, determine si el primero es un múltiplo del segundo.
package t02_02;

import java.util.Scanner;

public class Multiplos {

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        
        int numero1;
        int numero2;
        
        System.out.print("Escriba el primer entero: ");
        numero1 = sc.nextInt();
        
        System.out.print("Escriba el segundo entero: ");
        numero2 = sc.nextInt();
        
        if (numero1 % numero2 == 0)
            System.out.println(numero1 + " es multiplo de " + numero2);
        
        if (numero1 % numero2 != 0)
            System.out.println(numero1 + " no es multiplo de " + numero2);
    }
    
}

Ejercicio fdp01_t02_03: AritmeticaCirculo.java

Escriba una aplicación que reciba del usuario el radio de un círculo como un entero, y que imprima el diámetro, la circunferencia y el área del círculo.

// Ejercicio fdp01_t02_03: AritmeticaCirculo.java
// Aplicación que recibe el radio de un círculo como un entero, y que imprime 
// el diámetro, la circunferencia y el área del círculo
package t02_03;

import java.util.Scanner;

public class AritmeticaCirculo {

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        
        int radio;
        
        System.out.print("Escriba el radio del circulo: ");
        radio = sc.nextInt();
        
        System.out.println("El diametro es " + (2 * radio));
        System.out.println("La circunferencia es " + (2 * Math.PI * radio));
        System.out.println("El área es "+ (Math.PI) * Math.pow(radio, 2));
    }

}

5.- Ejercicio fdp01_t02_05: SeparacionDigitos.java

Escriba una aplicación que reciba del usuario un número compues- to por cinco dígitos, que separe ese número en sus dígitos individuales y los imprima, cada uno separado de los demás por tres espacios.

// Ejercicio fdp01_t02_05: SeparacionDigitos.java
// Aplicación que recibe del usuario un número compuesto por cinco dígitos, que separe ese número en sus dígitos 
// individuales y los imprima, cada uno separado de los demás por tres espacios.
package t02_05;

import java.util.Scanner;

public class SeparacionDigitos {

    public static void main(String[] args) {
        Scanner entrada = new Scanner(System.in);
        
        int numero;
        int unidad;
        int decena;
        int centena;
        int millar;
        int u_millar;
        
        System.out.print("Escriba un entero de cinco digitos: ");
        numero = entrada.nextInt();
        
        u_millar = numero / 10000;
        millar = numero / 1000;
        millar = millar % 10;
        centena = numero / 100;
        centena = centena % 10;
        decena = numero / 10;
        decena = decena % 10;
        unidad = numero / 1;
        unidad = unidad % 10;
        
        System.out.println(u_millar + "   " + millar + "   " + centena + "   " + decena + "   " + unidad);
    }
    
}
